# Security Policy

This repository primarily contains a specification and related documentation. If you discover a security issue in any tooling, scripts, or automation published from this repository, please report it responsibly.

## Reporting a Vulnerability

Preferred:

Fallback:
- Contact the maintainers listed in `MAINTAINERS.md`.

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Any suggested fix or mitigation

## Supported Versions

Security fixes are applied to the latest released major/minor line where practical. If no releases exist yet, fixes apply to the default branch.
