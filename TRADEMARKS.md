# Trademarks

This project name (“CFSE”) and any associated logos may be used to refer to the CFSE specification and this repository in an accurate, non-misleading way.

This repository does not grant trademark rights beyond nominative use (truthfully describing or referencing CFSE).

## Allowed Uses

- Referencing CFSE when describing compatibility or conformance.
- Using “CFSE” in academic or industry discussion with proper attribution.

## Not Allowed

- Implying endorsement by the CFSE maintainers without permission.
- Using CFSE branding to market or brand products or services (including training or certification programs) without permission.
- Using CFSE branding in a way that is confusingly similar to an official distribution.

## Questions

If you’re unsure whether a use is acceptable, contact the maintainers listed in `MAINTAINERS.md`.
